A high-impact, responsive hero section designed for modern brands. This project features a cinematic video background, glassmorphism UI elements, and sophisticated typography to create a compelling digital first impression.

> **Note:** The design concept and visual inspiration were sourced from **Pinterest**. The entire codebase, responsive logic, and animations were developed using **Gemini AI**.

## üöÄ About The Project

This landing page captures the essence of "Bold Ideas" through a minimalist yet powerful interface. It is designed to be fully responsive, ensuring a seamless experience across mobile devices, tablets, and desktop screens.

**Key Visual Elements:**
* **Full-Screen Video Background:** A looped, muted, and autoplaying video that sets an immersive tone immediately.
* **Glassmorphism Navbar:** A floating navigation bar with a backdrop blur effect, providing a modern, frosted glass look.
* **Shining CTA:** A "Get In Touch" button featuring a continuous, subtle light-shine animation to draw user attention.
* **Typography Hierarchy:** Large, serif headlines paired with clean sans-serif sub-text for maximum readability and style.

## ‚ú® Features

* **Fully Responsive:** Optimized layout that stacks elements vertically on mobile and adjusts spacing dynamically.
* **Video Integration:** HTML5 Video implementation with `autoplay`, `loop`, and `playsinline` support for iOS.
* **Tailwind CSS:** Utility-first styling for rapid UI development and easy customization.
* **Lucide Icons:** Lightweight and clean iconography.
* **Dynamic Animations:** Custom CSS keyframes for the button shine effect.

## üíª Installation

1.  Clone the repository:
    ```bash
    git clone [https://github.com/RAkhilKumar/Nexora-Ai-Landing-Page.git](https://github.com/RAkhilKumar/Nexora-Ai-Landing-Page.git)
    ```
2.  Navigate to the project directory:
    ```bash
    cd Nexora-Ai-Landing-Page
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```
